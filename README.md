# 性取向认知测试

一个帮助用户了解和探索性取向多样性的在线测试工具。

## 项目特点

- 专业的测试题库，包含30个精心设计的问题
- 现代化的UI设计，提供流畅的用户体验
- 详细的结果分析和个性化建议
- 支持多种性取向类型的识别和解释

## 技术栈

- React
- TypeScript
- Tailwind CSS
- React Router
